Projects

AstroWatch

An application that displays data from NASA's Near Earth Objects and Daily Image APIs. The app also collects user information and stores them in a database.

Technologies: React, MongoDB, Express, Node.js

AstroWatch is a Chingu group project. All three of us planned six one-week sprints on Trello using agile methodology. I had a blast building the front-end and connecting the app to our MongoDB database.

An application that stores notes.

Technologies: React, CSS Grid

Notes App is a solo project I designed using Figma. I learned about the "useState" hook in React and used CSS grid to make the app responsive.

June Birthday Calendar Generator

A Python program that creates a dictionary to which birthdays can be added or deleted from. The program prints how old each person in the final dictionary will be this year, as well as creates a birthday calendar using tkinter.

Technologies: Python, tkinter, datetime, openpyxl modules

This calendar generator was my final project for Stanford's Code In Place course.
